# Taxi Bavaria Website

Statische Kopie der Taxi-Bavaria-Website mit angepassten Kontaktdaten, Impressum und Fahrzeugflotte.

## Lokal testen

```bash
python3 -m http.server 4173
```

Danach im Browser `http://localhost:4173` öffnen.

## Upload

Alle Dateien im Projektordner können direkt in ein GitHub-Repository hochgeladen werden. Für GitHub Pages ist die Datei `404.html` enthalten, damit Unterseiten der Single-Page-App funktionieren.
